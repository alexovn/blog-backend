# Backend for blog on Nest.js

Backend for blog with use of Nest.js, Prisma ORM and PostgreSQL.

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
